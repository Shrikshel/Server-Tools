# st docker compose start-all

Start all stopped Docker containers

| Attributes       | &nbsp;
|------------------|-------------
| Alias:           | start

## Usage

```bash
st docker compose start-all
```

## Examples

```bash
st dc compose start-all
```

```bash
st docker compose start-all
```

```bash
st dc c start-all
```

```bash
st dc c start
```
